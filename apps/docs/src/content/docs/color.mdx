<h2 className="screen-reader-only">

<TextBlock>

Natura11y's color system makes it easier to keep color relationships consistent. Theme classes can be applied to full pages, sections, or individual components like cards and buttons.

</TextBlock>

<AccessibilitySpotlight>

Ensure your color combinations meet a <a href="https://colorable.jxnblk.com/" target="_blank">sufficient contrast</a> ratio.

Learn more about <a href="https://www.w3.org/WAI/WCAG21/Understanding/contrast-minimum.html" target="_blank">accessible contrast ratios</a>.

</AccessibilitySpotlight>

<Section id="theme-colors">

<TextBlock>

## Theme Colors: Your Principal Palette

Your principal theme colors include ***Primary***, ***Secondary***, ***Dark***, ***Light***, and ***Canvas***.

</TextBlock>

<PrincipalPalette />

<TextBlock>

These five colors produce utility classes for <a href="#theme-color-utilities">theme colors</a>, <a href="#background-color-utilities">background colors</a>, <a href="#text-color-utilities">text colors</a>, and <a href="/docs/border/#border-color-utilities">border colors</a>.

Natura11y treats ***canvas*** as a color, which serves as the default theme. The framework applies the canvas theme and its <a href="#correlated-colors">correlated colors</a> to the page by default.

</TextBlock>

</Section>

<Section id="correlated-colors">

<TextBlock>

## Correlated Colors

Every theme color has five correlated colors. These are used for text, borders, links, and instances of **confirm** and **warn**. Correlated colors work in isolation within their corresponding <a href="#theme-color-utilities">theme color utility</a> class.

Figure 1 (below) depicts the CSS variable declared for the ***primary*** theme on line 3. Lines 7-11 provide CSS variables for primary's correlated colors.

</TextBlock>

<FigureExample
    figureNumber="1"
    codeLanguage="scss"
    figureWidth="narrow"
    showCodeFrame={false}
    highlightedLines="3,7-11"
    code={`
        // Primary Theme Color

        --primary: #440381;

        // Primary Correlated Colors

        --primary-text: white;
        --primary-border: hsla(0, 0%, 0%, 0.25);
        --primary-link: currentColor;
        --primary-confirm: #16db93;
        --primary-warn: #ff73aa;
    `}
/>

<TextBlock>

For a descriptive list of all colors available, jump over to the <a href="/docs/customization/#cp_color" target="_blank">CSS variable reference guide</a>.

</TextBlock>

</Section>

<TextBlock>

## Theme Color Utilities

Natura11y's theme color utility classes are one of its most useful features. Each theme color utility provides predefined colors for <strong>background</strong>, <strong>text</strong>, <strong>borders</strong>, and <strong>links</strong> (within paragraphs). Theme color utilities also provide colors for instances of <strong>confirm</strong> and <strong>warn</strong>.

Figure 2 (below) provides the five theme color utilities:

</TextBlock>

<FigureExample
    figureNumber="2"
    figureWidth="narrow"
    code={`
        <div class="theme-primary">...</div>
        <div class="theme-secondary">...</div>
        <div class="theme-dark">...</div>
        <div class="theme-light">...</div>
        <div class="theme-canvas">...</div>
    `}
/>

<TextBlock>

You may apply theme colors to any containing block-level element within the DOM. Consider the following five cards, each colored with a different theme color utility.

</TextBlock>

<ThemeColors />

<TextBlock>

In Figure 3, notice the `.theme-primary` utility. No other element within the card needs its color specified. Each theme color utility applies its <a href="#correlated-colors">correlated colors</a> to descendants.

</TextBlock>

<FigureExample
    figureNumber="3"
    figureWidth="medium"
    markedText="theme-primary"
    code={`
        <div class="card theme-primary border-radius-2">

            <div class="card__head border-bottom">
                <p>
                    <strong>Theme: Primary</strong>
                </p>
            </div>

            <div class="card__body">
                <h2 class="h4">
                    Rainbow lorikeet
                </h2>
                <p>
                    The rainbow lorikeet is a species of parrot found in Australia. It is common along the eastern seaboard, from northern <a href="#">Queensland</a> to <a href="#1">South Australia</a>. Its habitat is rainforest, coastal bush and woodland areas.
                </p>
            </div>

            <div class="card__foot border-top">
                <ul class="nav nav--horizontal justify-content-between">
                    <li>
                        <a href="#1">Search Images</a>
                    </li>
                    <li>
                        <a class="button button--outline border-radius-pill" href="#1">
                            Visit Wikipedia
                        </a>
                    </li>
                </ul>
            </div>

        </div>
    `}
/>

<TextBlock>

### Links

Refer to Figure 1 (above), on line 9. The correlated link color (e.g. <code>--primary-link</code>) value sets the color of generic links (<code>&lt;a&gt;</code> tags) within paragraphs (<code>&lt;p&gt;</code> tags). This is not only convenient, it keeps paragraph links visible and accessible.

</TextBlock>

<LinkColors />

<TextBlock>

Beyond generic links, a <a href="#text-color-utilities">text color utility</a> named <code>.text-color-link</code> is available. Figure 4 uses <code>.text-color-link</code> to color the buttons with the current theme's link color.

</TextBlock>

<FigureExample
    figureNumber="4"
    markedText="text-color-link"
    code={`
        <div class="display-flex gap-2 justify-content-center text-color-link">

            <a class="button button--outline" href="#1">
                Sign In
            </a>

            <a class="button button--icon-only" href="#1" aria-label="Language">
                <span class="icon icon-language" aria-hidden="true"></span>
            </a>

            <a class="button button--icon-only" href="#1" aria-label="Search">
                <span class="icon icon-search" aria-hidden="true"></span>
            </a>

        </div>
    `}
>
    <LinkButtons />
</FigureExample>

<TextBlock>

### Confirm and Warn

In Figure 1 (above), on lines 10 and 11, you will find declarations for ***confirm*** and ***warn*** color values. Natura11y allows you to specify confirm and warn values for each theme color!

</TextBlock>

<FeedbackColors />

<TextBlock>

Instances of *confirm* and *warn* appear in <a href="/docs/alert/">alert</a> and <a href="/docs/form/">form</a> elements.

</TextBlock>

<TextBlock>

### Infinitely Nestable

You can nest theme color utilities throughout the DOM.

</TextBlock>

<InfinitelyNestable />

### Breakpoint Specific

Apply a theme color at a specific breakpoint. Chain a breakpoint modifier, such as `--lg` for the large breakpoint, to the color utility. Figure 5 applies `.theme-primary--lg` to the parent of the navigation.

</TextBlock>

<FigureExample
    figureNumber="5"
    markedText="theme-primary--lg"
    code={`

                </li>
                <li>
                    <a href="#1">Link</a>
                </li>
                <li>
                    <a href="#1">Link</a>
                </li>
            </ul>

        </div>
    `}
>
    <BreakpointNav />
</FigureExample>

<TextBlock>

On smaller viewports, the example uses the default ***canvas*** theme. It switches to the ***primary*** theme when the viewport reaches the large breakpoint. Scale your browser horizontally to see this change occur.

</TextBlock>

<Section id="subtle-fill">

<TextBlock>

## Subtle Fill

The `.subtle-fill-1` and `.subtle-fill-2` classes add light background tints to any element, using a percentage of the element's text color (`currentColor`).

- `.subtle-fill-1` applies a background color that is 5% of the current text color.
- `.subtle-fill-2` applies a background color that is 10% of the current text color.

In the card examples below, we apply a subtle fill to the card footer with these variations.

</TextBlock>

<FigureExample
    figureNumber="6"
    markedText="subtle-fill-1"
    code={`
        <div class="card theme-dark">

            <div class="card__body">
                <!-- ... -->
            </div>

            <div class="card__foot subtle-fill-1">
                <!-- ... -->
            </div>

        </div>
    `}
>
    <SubtleFill />
</FigureExample>

<TextBlock>

Notice how subtle fills appear differently depending on the theme: darker on the canvas theme and lighter on the dark theme.

</TextBlock>

</Section>

Natura11y's text color utilities apply a text color to specific elements.

Figure 8 applies the primary theme to a containing element. The third button uses `.text-color-secondary`, which overrides the inherited theme text color.
