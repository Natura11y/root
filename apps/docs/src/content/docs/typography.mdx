Natura11y offers a wide range of customizable typography styles. Out-of-the-box, font sizes are set using a <LinkOpenNew LinkUrl='https://type-scale.com' LinkText='type scale' />. <Section id="banner-headline">

<TextBlock>

## Banner Headline

The `.banner-headline` utility is named after the large headlines in newspapers that stretch across the top of the front page. This utility is useful for creating a header that is larger or visually distinct from the usual `<h1>` tag.

</TextBlock>

<FigureExample
	figureNumber="1"
  code={`
  <h1 class="banner-headline">
    Banner Headline
  </h1>
  `}
>
    <BannerHeadline />
</FigureExample>

</Section>

<TextBlock>

## Section Headings

Natura11y provides six levels of section headings, `<h1>` through `<h6>`. Styles for each level are consistent for a tag header (e.g. `<h1>`) and a utility class header (e.g. `.h1`).

</TextBlock>

<FigureExample
	figureNumber="2"
	code={`
	<h1>Header 1</h1>
	<p class="h1">Header 1</p>

	<h2>Header 2</h2>
	<p class="h2">Header 2</p>

	<h3>Header 3</h3>

<Section id="natura11y-type-scale">

<TextBlock>

## Natura11y's Type Scale

A type scale helps maintain consistent, readable font sizes across your projects and screen sizes. In Natura11y, font sizes follow a simple type scale that you can easily adjust with two properties.

The first property, `--base-font-size`, controls the smallest font size. The second property, `--type-scale`, sets the multiplier used to create larger font sizes. Each size is a multiple of the one before it, based on the scale value.

Use the <a href="/type-scale/">Type Scale Visualizer</a> to test different base font sizes and type scale values.

For more details, check out the <a href="#related-custom-properties">Related Custom Properties</a> section at the end of this page, which explains how font sizes relate to each other.

</TextBlock>

<TextBlock>

### Responsive Fonts

You can easily make fonts responsive by using a different type scale inside a `@media` or `@container` query (see Figure 8).

</TextBlock>

<FigureExample
	figureNumber="8"
	figureWidth="narrow"
	codeLanguage="scss"
	markedText="--type-scale: 1.125"
	code={`
  /* Adjust the type scale at a specific screen size */

  @media (max-width: 768px) {
    :root {
      --type-scale: 1.125
    }
  }
	`}
/>

</Section>
