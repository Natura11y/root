## Box Shadow Utilities

Natura11y's box shadow utilities come in three sizes. Each size is represented by a number: <em>1</em> for <strong><em>small</em></strong>, <em>2</em> for <strong><em>medium</em></strong>, and <em>3</em> for <strong><em>large</em></strong>.

</TextBlock>

<FigureExample
    figureNumber="1"
    code={`
        <div class="box-shadow-1">...</div>
        <div class="box-shadow-2">...</div>
        <div class="box-shadow-3">...</div>
    `}
>
    <BoxShadow />
</FigureExample>

<TextBlock>

### Breakpoint Modifiers

Box shadow utilities accept <a href="/docs/breakpoint/#breakpoint-modifiers">breakpoint modifiers</a>.

</TextBlock>

<FigureExample
    figureNumber="2"
    code={`
        <!-- With breakpoint modifier -->

        <div class="box-shadow-1--lg 👈">...</div>
    `}
/>

</Section>

<Section id="drop-shadow-utilities">

<TextBlock>

## Drop Shadow Utilities

Drop shadow utilities use the CSS <LinkOpenNew LinkUrl='https://developer.mozilla.org/en-US/docs/Web/CSS/filter-function/drop-shadow' LinkText='filter: drop-shadow()' /> function. Unlike box shadows, drop shadows follow the shape of the element, making them ideal for transparent images, SVGs, and cutout shapes.

Drop shadow utilities come in three sizes: <em>1</em> for <strong><em>small</em></strong>, <em>2</em> for <strong><em>medium</em></strong>, and <em>3</em> for <strong><em>large</em></strong>.

</TextBlock>

<FigureExample
    figureNumber="3"
    code={`
        <div class="drop-shadow-1">...</div>
        <div class="drop-shadow-2">...</div>
        <div class="drop-shadow-3">...</div>
    `}
>
    <DropShadow />
</FigureExample>

<TextBlock>

### Breakpoint Modifiers

Drop shadow utilities accept <a href="/docs/breakpoint/#breakpoint-modifiers">breakpoint modifiers</a>.

</TextBlock>

<FigureExample
    figureNumber="4"
    code={`
        <!-- With breakpoint modifier -->

        <div class="drop-shadow-1--lg 👈">...</div>
    `}
/>

</Section>
