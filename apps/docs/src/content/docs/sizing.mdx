## Width Utilities

Natura11y provides the following width utilities. Number values represent a percentage. For example, the <code>.width-50</code> utility makes its selector 50% of its parent. The <code>.width-100vw</code> and <code>.width-auto</code> utilities provide viewport-width and automatic width options.

</TextBlock>

<FigureExample
    figureNumber="1"
    code={`
        <div class="width-10">...</div>
        <div class="width-20">...</div>
        <div class="width-25">...</div>
        <div class="width-30">...</div>
        <div class="width-40">...</div>
        <div class="width-50">...</div>
        <div class="width-60">...</div>
        <div class="width-70">...</div>
        <div class="width-75">...</div>
        <div class="width-80">...</div>
        <div class="width-90">...</div>
        <div class="width-100">...</div>
        <div class="width-100vw">...</div>
        <div class="width-auto">...</div>
    `}
>
    <Width />
</FigureExample>

<TextBlock>

### Breakpoint Modifiers

Width Utilities accept <a href="/docs/breakpoint/#breakpoint-modifiers">breakpoint modifiers</a>.

</TextBlock>

<FigureExample
    figureNumber="2"
    markedText="width-75--lg"
    code={`
        <!-- With breakpoint modifier -->

        <div class="width-75--lg">...</div>
    `}
/>

<Section id="height-utilities">

<TextBlock>

## Height Utilities

Natura11y provides height and minimum height utilities for full-height elements:

</TextBlock>

<FigureExample
    figureNumber="3"
    code={`
        <div class="height-100">...</div>
        <div class="min-height-100">...</div>

        <!-- 100% height of viewport -->
        <div class="height-100vh">...</div>
        <div class="min-height-100vh">...</div>
    `}
>
    <Height />
</FigureExample>

<TextBlock>

### Breakpoint Modifiers

Height and minimum height utilities accept <a href="/docs/breakpoint/#breakpoint-modifiers">breakpoint modifiers</a>.

</TextBlock>

<FigureExample
    figureNumber="4"
    markedText={['height-100--lg', 'min-height-100--lg', 'height-100vh--lg', 'min-height-100vh--lg']}

        <div class="height-100--lg">...</div>
        <div class="min-height-100--lg">...</div>
        <div class="height-100vh--lg">...</div>
        <div class="min-height-100vh--lg">...</div>
    `}
/>
