<Section id="breakpoint-modifiers">

<TextBlock>

### Breakpoint Modifiers

Aspect ratio utilities can use <a href="/docs/breakpoint/#breakpoint-modifiers">breakpoint modifiers</a> to adjust sizes at different screen widths.

</TextBlock>

<FigureExample
  figureNumber="2"
  code={`
  <!-- With breakpoint modifier -->

  <div class="aspect-ratio-16x9--lg">...</div>
  `}
/>

</Section>

<Section id="embedding-video">

<TextBlock>

## Embedding Video

Aspect ratios are useful for embedding videos on a page. Here is an example:

</TextBlock>

<FigureExample
  figureNumber="3"
  highlightedLines="2"
  code={`
  <iframe
    class="aspect-ratio-16x9"
    src="..."
    frameborder="0"
    allow="autoplay;"
    allowfullscreen>
  </iframe>
  `}
>
  <iframe
    class="aspect-ratio-16x9 background-color-dark"
    src="https://player.vimeo.com/video/261715638"
    title="Arches National Park"
    style={{ border: 0 }}
    allow="autoplay; fullscreen"
    allowfullscreen>
  </iframe>
</FigureExample>
