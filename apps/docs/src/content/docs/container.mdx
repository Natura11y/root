## The Container Utility

The container utility (`.container`) applies consistent left and right padding. A container without a <a href="#width-variations">width variation utility</a> spans the full width of its parent.

</TextBlock>

<Container />

<FigureExample
    figureNumber="1"
    figureWidth="medium"
    markedText="container"
    code={`
        <div class="container">...</div>
    `}
/>

## Breakpoint Modifiers

The container utility accepts <a href="/docs/breakpoint/#breakpoint-modifiers">breakpoint modifiers</a>. Use them to defer the container padding until a specific breakpoint.

Consider the HTML in Figure 2. The element has no padding below the large breakpoint. At the large breakpoint and above, it gains the standard container padding.

</TextBlock>

<ContainerBreakpoint />

<FigureExample
    figureNumber="2"
    figureWidth="medium"

        <div class="container--lg">...</div>

<Section id="width-variations">

<TextBlock>

## Width Variations

Width variation utilities come in three sizes: ***narrow***, ***medium***, and ***wide*** (Figure 3). Apply them to containers or use them independently.

When present on `.container` elements, width variation utilities (e.g. `narrow`) center horizontally within their parent.

</TextBlock>

<WidthVariations />

<FigureExample
    figureNumber="3"
    figureWidth="medium"
    markedText={['narrow', 'medium', 'wide']}
    code={`
        <div class="container narrow">...</div>
        <div class="container medium">...</div>
        <div class="container wide">...</div>
    `}
/>

<TextBlock>

### Breakpoint Modifiers

Width variation utilities accept <a href="/docs/breakpoint/#breakpoint-modifiers">breakpoint modifiers</a>.

Consider the HTML in Figure 4. The container starts at the **narrow** width. It then changes to the **medium** width at the large breakpoint. Finally, the container changes to the **wide** width at the extra-large breakpoint.

</TextBlock>

<BreakpointModifier />

<FigureExample
    figureNumber="4"
    figureWidth="medium"
    markedText={['narrow', 'medium--lg', 'wide--xl']}
    code={`
        <!-- With breakpoint modifiers -->

        <div class="container narrow medium--lg wide--xl">...</div>
    `}
/>

</Section>
