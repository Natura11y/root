    Introduction
</h2>

<TextBlock>

Natura11y's lightbox opens images and video in an overlay. It works with:

- High-resolution images
- Local video files
- YouTube videos
- Vimeo videos

</TextBlock>

<AccessibilitySpotlight>

When a lightbox is open, the <kbd>Tab</kbd> key moves through the lightbox controls without leaving the overlay. The <kbd>Left Arrow</kbd> and <kbd>Right Arrow</kbd> keys move between items, and <kbd>Esc</kbd> closes the lightbox.

When the lightbox closes, focus returns to the button that opened it.

</AccessibilitySpotlight>

</Section>

<Section id="lightbox-thumbnail-buttons">

<TextBlock>

## Lightbox Thumbnail Buttons

Use a `<button>` with the required `data-lightbox` attributes to open a lightbox.

Add `.lightbox-thumbnail` when the button contains an image or video thumbnail. Figure 1 shows an image lightbox and a video lightbox.

</TextBlock>

<FigureExample
  figureNumber="1"
  highlightedLines="3-6"
  code={`
  <!-- Image lightbox -->
  <button
    class="lightbox-thumbnail"
    data-lightbox="image"
    data-lightbox-src="/path/to/image.jpg"
    data-lightbox-caption="..."
  >
    <img
      src="/path/to/thumbnail.jpg"
      alt="..."
    />
  </button>

  <!-- Video lightbox -->
  <button
    class="lightbox-thumbnail"
    data-lightbox="video"
    data-lightbox-src="/path/to/video.mp4"
    data-lightbox-caption="..."
  >
    <img
      src="/path/to/thumbnail.jpg"
      alt="..."
    />
  </button>
  `}
>
  <LightboxThumbnailButtons />
</FigureExample>

<TableLineDescBlock targetFigure="1">

<TableLineDescRow targetFigure="1" lines="3">

The `.lightbox-thumbnail` class is optional. It adds the thumbnail overlay icon.

</TableLineDescRow>

<TableLineDescRow targetFigure="1" lines="4">

The `data-lightbox` attribute is required. Use `image` for images and `video` for videos.

</TableLineDescRow>

<TableLineDescRow targetFigure="1" lines="5">

The `data-lightbox-src` attribute sets the media source. For image lightboxes, it can be omitted when the child image `src` should be used.

</TableLineDescRow>

<TableLineDescRow targetFigure="1" lines="6">

The `data-lightbox-caption` attribute is optional. It is used to add a caption to the lightbox.

</TableLineDescRow>

</TableLineDescBlock>

</Section>

<TextBlock>

## Custom Lightbox Buttons

Any button can open a lightbox when it includes the required `data-lightbox` attributes. Figure 2 uses a standard button.

</TextBlock>

<FigureExample
  figureNumber="2"
  highlightedLines="3-5"
  code={`
  <button
    class="button theme-primary width-100"
    data-lightbox="video"
    data-lightbox-src="https://www.youtube.com/embed/XXXXXXXXXX"
    data-lightbox-caption="YouTube video"
  >
    Open YouTube Video
  </button>
  `}
>
  <LightboxCustomButtons />
</FigureExample>

## Lightbox Gallery

Natura11y's JavaScript automatically collects every `[data-lightbox]` trigger on the page into one gallery. When more than one lightbox item exists, visitors can move through the gallery with the *Previous* and *Next* buttons or the <kbd>Left Arrow</kbd> and <kbd>Right Arrow</kbd> keys.
