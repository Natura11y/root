<h2 className="screen-reader-only">
    Introduction
</h2>

<TextBlock>

The nested navigation component provides a structured, multi-level sidebar navigation. It supports up to three levels of hierarchy, with indentation and font size communicating depth.

</TextBlock>

<AccessibilitySpotlight>

Wrap the nested nav in a `<nav>` element with a descriptive `aria-label` to identify it as a navigation landmark.

Use `aria-current="true"` on a parent section when a deeper page is selected, and `aria-current="page"` on the link that represents the current page.

</AccessibilitySpotlight>

</Section>

---

<Section id="nav-nested">

<TextBlock>

## Nested Navigation

Add `.nested-nav` to a `<ul>` element. In Figure 1, the level-one link marks the active section, and the nested link marks the current page.

</TextBlock>

<FigureExample
  figureNumber="1"
  markedText={['nested-nav', 'aria-current="true"', 'aria-current="page"']}
  code={`
  <nav>

    <ul class="nested-nav">
      <li>
        <a href="#">Overview</a>
      </li>
      <li>
        <a href="#" aria-current="true">American Robin</a>
        <ul>
          <li>
            <a href="#" aria-current="page">Nesting</a>
          </li>
          <li>
            <a href="#">Diet</a>
          </li>
          <li>
            <a href="#">Migration</a>
          </li>
        </ul>
      </li>
      <li>
        <a href="#">Eastern Bluebird</a>
        <ul>
          <li>
            <a href="#">Identification</a>
            <ul>
              <li>
                <a href="#">Plumage</a>
              </li>
              <li>
                <a href="#">Song</a>
              </li>
            </ul>
          </li>
          <li>
            <a href="#">Habitat</a>
          </li>
        </ul>

      <li>
        <a href="#">Conservation</a>
      </li>
      <li>
        <a href="#">Resources</a>
      </li>
    </ul>
    
  </nav>
  `}
>
  <Default />
</FigureExample>
