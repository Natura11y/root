<TextBlock>

Natura11y uses the `@layer` CSS <LinkOpenNew LinkUrl='https://developer.mozilla.org/en-US/docs/Web/CSS/@layer' LinkText='at-rule' /> to organize framework styles by purpose.

</TextBlock>

</Section>

<Section id="cascade-layers">

<TextBlock>

## Cascade Layers

Natura11y includes three cascade layers: ***base***, ***component***, and ***utility***. These layers are declared first so every partial follows the same cascade order.

</TextBlock>

<FigureExample
  figureNumber="1"
  figureWidth="narrow"
  codeLanguage="css"
  showCodeFrame={false}
  code={`
  /* Natura11y's CSS Layer Declaration */

  @layer base, component, utility;
  `}
/>

</Section>

---

<Section id="the-base-layer">

<TextBlock>

## The Base Layer

The `base` layer contains element selectors and targeted resets. Figure 2 shows a shortened example from the framework's base styles.

</TextBlock>

<FigureExample
  figureNumber="2"
  figureWidth="narrow"
  codeLanguage="css"
  showCodeFrame={false}
  code={`
  /* Base Layer Styles */

  @layer base {

    *,
    *::before,
    *::after {
      box-sizing: border-box;
    }

    html {
      scroll-behavior: var(--html-scroll-behavior);
    }

    body {
      min-width: var(--body-min-width);
      max-width: var(--body-max-width);
    }
    
  }
  `}
/>

</Section>

---

<Section id="the-component-layer">

<TextBlock>

## The Component Layer

The `component` layer contains class-based component styles. Figure 3 shows a shortened button example.

</TextBlock>

<FigureExample
  figureNumber="3"
  figureWidth="narrow"
  codeLanguage="css"
  showCodeFrame={false}
  code={`
  /* Component Layer Styles */

  @layer component {
    
    .button {
      display: inline-flex;
      justify-content: center;
      align-items: center;

      border-radius: var(--button-border-radius);
    }
  
  }
  `}
/>

</Section>

---

<Section id="the-utility-layer">

<TextBlock>

## The Utility Layer

The `utility` layer contains reusable classes that can be added to any element. Figure 4 shows shortened display and border-radius examples.

</TextBlock>

<FigureExample
  figureNumber="4"
  figureWidth="narrow"
  codeLanguage="css"
  showCodeFrame={false}
  code={`
  /* Utility Layer Styles */

  @layer utility {

    .display-flex {
      display: flex;
    }

    .border-radius-2 {
      border-radius: var(--border-radius-2);
    }
  
  }
  `}
/>

</Section>
