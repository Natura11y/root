## Modernized Styles with Natura11y

Natura11y uses the latest, browser-friendly CSS specifications.

Key features of Natura11y's CSS include:

</TextBlock>

<ul class="container narrow margin-y-4 grid gap-2">
  <FeatureCard title="CSS Variables">
    <p>You can customize Natura11y's default styles with CSS variables, also known as custom properties. Each CSS variable matches a specific Sass module.</p>
    <ul class="nav text-color-link">
      <li>
        <a href="/docs/customization/">Customization</a>
      </li>
    </ul>
  </FeatureCard>

  <FeatureCard title="Cascade Layers">
    <p>Natura11y's CSS uses cascade layers to make styling more predictable and semantic, reducing the need for the <code>!important</code> flag.</p>
    <ul class="nav text-color-link">
      <li>
        <a href="/docs/layer/">Using Cascade Layers</a>
      </li>
    </ul>
  </FeatureCard>

  <FeatureCard title="Logical Properties">
    <p>Natura11y uses CSS logical properties for directional values. These properties are essential for supporting right-to-left languages.</p>
    <ul class="nav text-color-link">
      <li>
        <a href="/docs/rtl/">RTL and Logical Properties</a>
      </li>
    </ul>
  </FeatureCard>

  <FeatureCard title="Typography Scale">
    <p>Natura11y's default font sizes are related from smallest to largest, making it easy to create balanced typography with one or two variables.</p>
    <ul class="nav text-color-link">
      <li>
        <a href="/docs/typography/">Natura11y's Type Scale</a>
      </li>
    </ul>
  </FeatureCard>
</ul>

</Section>
